Text Encryptor

Images

App to encrypt and decrypt messages choosing any of the 8 codes

  • Enter any message to encrypt and choose encryption code
  • Choose the same code to decrypt
  • Feature to Copy encrypted text by long pressing
  • Feature to Paste copied text onto text for decryption
  • Can use any characters
  • It only encryptes the alphabetical letters and doesnt affect others
  • Share your code with your friend to encrypt and decrypt your messages!

App to see and save all the events happening in college and much more !

  • Anyone in permission can add/delete the events from the server

  • Offline view has recycler view which looks preety neat!

  • Various about and settings tabs to explain any steps
